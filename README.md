## 玩法介绍

《抓大鸽》是一款翻盖三消休闲游戏。棋盘上摆满扣着的盖板，点击翻开即可随机揭晓隐藏的鸽子。同类鸽子在横向或纵向形成 **3 连及以上**时自动消除并得分。

每局共有 **30 次**翻盖机会，用尽后按当前总分结算。单局为固定尺寸棋盘（7×9），以尽可能高的总分为目标；累计得分越高，后续揭晓时**咕咕精**（扣分）出现概率越大。

### 鸽子品类

| 名称 | 分数 | 说明 |
|------|------|------|
| 普通鸽 | +10 | 最常见 |
| 呆呆鸽 | +20 | — |
| 咕咕精 | -30 | 扣分陷阱 |
| 金鸽子 | +100 | 高分奖励 |
| 大鸽王 | +500 | 翻出后清除**全部已翻开**格子并额外结算分数 |

### 特殊规则

- **三连消除**：仅统计已翻开的格子；消除后原位补充新的带盖格子，若再次形成三连则**连锁**继续消除。
- **大鸽王**：翻出后播放专属动画，清除全部已翻开格子（含自身）；得分 = 鸽王基础分 + 其余被清除格子分数绝对值之和。未翻开的格子中始终至少藏有 1 只大鸽王。
- **全盘翻开**：若在剩余翻盖次数内将全部格子翻开，会重新布局为带盖新鸽子，可继续在本局剩余次数内游玩。

### 操作

| 操作 | 说明 |
|------|------|
| 点击盖板 | 翻开格子，揭晓鸽子 |
| 自动消除 | 同类横/纵 ≥3 连续时立即消除 |
| 音效开关 | 右上角按钮切换静音 |
